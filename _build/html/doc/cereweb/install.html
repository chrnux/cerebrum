Installation instructions for Cereweb (Cerebrum web interface)¶

Requirements¶

  • Python 2.4 or newer
  • Cheetah 2.* http://www.cheetahtemplate.org/
  • CherryPy 2.2.* http://www.cherrypy.org/
  • cjson 1.*

Install¶

Run ”./configure” from your cerebrum cvs-root to generate makefiles.

Run “make” or “gmake” in your cereweb directory to generate css and python files from the cheetah templates.

Run “python cereweb.py” from your cereweb directory to start the server.

Configuration¶

cherrypy.conf * configurartion for cherrypy

cereweb.conf * configuration for cereweb

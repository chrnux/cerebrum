Configuring and running Cerebrum¶

This document aims to describe the procedures necessary for running and administering Cerebrum within an organization. The routines for tailoring, bootstraping, logging, cleaning, upgrading etc. are described from the administrators point of view. This document will in addition contain some of the simpler troubleshooting procedures.
